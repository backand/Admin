Category

Category view holds the names of the tabs (categories) that help handle and sort the parameters on each view.

For example, in order to set a parameter on a certain view, under one of the tabs, the following actions should be performed:

  • open Views and find the relevant view.
  • It's possible to edit on-grid or open the edit dialog
  • Under Category choose the relevant category (Tab) that is required.
  • In case that the tab does not appear on the drop down list, it should be added to the list:
    • Open Admin - Category.
    • Click on New and add the category.
  • After Commiting, the field will appear under the relevant category.
