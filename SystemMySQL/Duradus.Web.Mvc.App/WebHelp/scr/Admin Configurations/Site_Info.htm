Site Info

Site Info contain information that present the client's details - company name, logo...

Presenting the customer's data depends on following the actions below -

  • Define the Information Parameters:
  • Company
  • Product
  • Version
  • Logo – Can upload a picture file
  • Logo Href – can add link when clicking on the Logo.
  • Log On Path – Determine the Log On Path
  • Check the check boxes of the data you would like to present:
  • Show Company
  • Show Product
  • Show Version
  • Show Logo
  • Save & Commit and close the dialog.
  • Go to Admin - Database and click on Edit.
  • On Site Info, choose the company name, and Save & commit again.
  • When refreshing the browser, the site info data would appear at the top of the system.
