Menus

There are 2 differnet sorts of menus on Durados:

Menu - Defined on View's parameters - Display - each view can be displayed under a menu.

This menu contain only 1 level.

Special Menu -

In order to display the view, the following actions should be taken:

1. Open Admin-Database

2. First step is to locate the menu which will display the view. Click on Menus – the existing menus options appear.

3. It’s possible to choose existing menu or create a new one

a. Creating New Menu –

1. Click on New (or Duplicate).

2. The required values are:

1. (Required field) Name (could be any name – it will be the name that will be displayed)

2. Order (from smallest number to the biggest)

4. Under the menu there are 2 options:

a. Create \ Choose sub-menu - Creating Sub-Menu is identical process to creating Menu

b. Create Link to display the view –

1. Click on Link – the list of existing links open.

2. Click on New \ Duplicate

3. Therequires values are:

1. (Required field) Title – The view name that will be displayed on system.

2. (Required field) View Name - (actual name of the view – should be identical to name that appears on DB and Admin-Views menu, under Name)

3. Order

4. Workspace – in order to apply permissions for roles, the suitable workspace whould be selected.

Choosing Save & Commit will create the new view display
