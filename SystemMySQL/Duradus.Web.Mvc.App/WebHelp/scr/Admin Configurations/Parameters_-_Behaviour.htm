Behaviour tab display the following parameters:

 

Inputs & Outputs - Parameters that define the option to connect with excel and printing rows:

  • Export to Excel: Checking that parameter would display Export option on the upper menu, and would allow exporting all the data on screen to Excel file, as explained on Import & Export section.

  • Import from Excel: Checking that parameter would display Import on the upper menu, and would allow importing excel files that contain data to the view, as explained on Import & Export section.

  • Print: Checking that option would display Print option on the upper menu, and would allow printing data from the view.

Editing Parameters - those parameters define some of the editing options the admin is allowed to perform.

Below are explanations of the behaviour when the parameter is checked. :

  • Allow Create: Allow creating new row on that view.

  • Allow Edit: Allow editing an existing row on that view.

  • Allow Duplicate: Allow duplicating an existing row on that view, in order to create new row.

  • Allow Delete: Allow deleting existing row.

  • Grid Editable: When this parameter is checked, it allow the user to edit data on-grid, without the need to open dialog screen.

  • Grid Editable Enabled: When this parameter is checked, it allow edit on-grid as a default status.


    When the parameter 'Grid Editable' is unchecked, this parameter does not affect the view.

  • Save History: Whe checked all the history of the view would be seved under History Log.

  • Multi Select?: Allow to check several rows (with Ctrl key), in order to perform actions on more then one row in an action.

  • History Notify List:?

  • Max Sub Grid Height: Determine the sub-grid height in pixels.
