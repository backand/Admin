Import & Export

Durados allows to connect with Excel - to import data from excel and to export data as well.

  • Export - Allow exporting any amount of rows from a view - from single row to the whole view -
    1. Mark all the rows you like to export.
    2. Click Export on the upper command menu.
    3. A file named <View_Name>.csv would be created and can be viewed by Excel.

  • Import - Allow importing data from Xls file (csv file) -
    1. Create file with the relevamnt data that should be inserted to the view.
    2. There are several options to import rows:
      • edit existing rows.
      • Add new rows
      • perform both - edit+add
    3. Click on Import - the rows would be added\edited to the view rows.
    4. In case that something in the file is incorrect - row missing, data that cannot be updated... - a notification message ould appear.
    5. Errors can be viewed by downloading the file the have been imported - the new file contains tab called Errors, that points to the rows that got errors on them, and explain what is the problem.
    6. After fixing the errors import can be performed again, successfully.
