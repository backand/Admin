Parameters - General

The General parameters are the ones that appear above all the tabs.

Those parameters would be always display when the Edit \ Add dialog is open, no matter which tab have been selected:

  • Name (Required Field) - A ctual name of the view. Should be identical to name that appears on DB.

  • Display Name(Required field) - C ould be any name. It will be the name that will be displayed.

  • Workspace - define the workspace in order to determine the permissions allowed on the view.

  • Descrtiption / Tooltip – text box which explains the view content. The text would be displayed when standing on the view
