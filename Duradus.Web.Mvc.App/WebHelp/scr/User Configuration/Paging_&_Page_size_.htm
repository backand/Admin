Paging & Page size

Durados allow navigation between pages and define the number of rows to be displayed on each page, in case that the view contains large amount of rows.

Paging definitions are done by the page bar, at the bottom of the view:

  • Changing Page size - Allow changing the amount of rows displayed on every page (Default page size is 20 rows) -
    1. Choosing different values on the Show drop down, allow displaying different amount of view on page, between 5 and 1,000.
    2. Any value selected would define the maximum amount of rows to display - if there are less rows, all rows would be dispaled.
    3. Leaving the view will reset the defult page size & paging.
    4. Changing the default value is possible via Admin Configuration.

  • Paging Backward & Forward - Allow navigating between the view pages -
    1. Click on the arrows allow navigate between the pages. each click will move it by 1 page.
    2. In case that the first page is displayed, moving backwards is impossible. In case the last page is displayed. moving forward is impossible.
    3. It is also possible to write the page number on the box, where the page number is display - the requested page would appear, if it exist.

  • Paging to Last / First page - Allow jumping to first \ last page -
    1. Clicking on the last / first arrows allow navigating straight to the beginning or the end of the view.
    2. It is also possible to write the first\last page's number on the box, where the page number is display - the requested page would appear.
