Duraos logs any change that have been performed on any view and any row.

Any change can be tracked down by looking at the Histort Log, which locate on the upper commands menu.

History can be viewed for single row, or for the whole view:  

  • History for a single row - Display all the changes that been made to a single row, presenting the date of change, old & new value and who is the user who performed the change:
    1. Check mark a row and click on History on the commands menu.
    2. On the history log that opens, all the changes appear.
    3. Viewing the history of deleted rows is not impossible when viewing single row's history.

  • History for all rows - Display all the changes made on the current view, respecting all rows.
    1. Click on History on the commands menu (do not mark any row).
    2. Changes of all rows appear on the log that opens.
    3. Viewing History Log for all rows also present the rows that have been deleted, unlike the history displayed to a single row.
