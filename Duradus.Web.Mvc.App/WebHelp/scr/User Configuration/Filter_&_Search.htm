Duraods contain some options to Sort, Filter and to perform Search on the view:

  • Sort - When openning a view it is sort by a default columns. The user can change that sort on that entry.
    Changing the sort permanently can be done by Admin. 
    1. Clicking on column header will sort by that column.
    2. One click will sort the values Ascending. Another click will sort the values descending.
    3. When leaving the view and re-entering it, the sort will return to its default sort Changing the default sort is an Admin option - ).
       
  • Filter  - In order to filter, please perform the following actions:
    1. On top of the column headers there are boxes that contain data according to the column type - date, text, drop-down, Yes\No...
    2. When filtering, insert data into the requested column.
    3. On the top commands menu there is Filter with 2 options - Apply & Clear.
    4. In order to filter click on Apply - the data filter as requested.
    5. Note that Filter on the commands menu changed color to Red - Whenever the view have been filtered, it will be colored Red.
    6. Click Clear in order to remove filter - Filter changed to black.
       
  • Multi-Filter - there is also an option to perform multi-filter on the view.
    1. Follow the actions described on Filter section above.
    2. It is possible to filter by as many columns as desired, by inserting the requested data into the boxes above column's headers.
    3. The rows to be displayd would be the ones that contain the requested data from all filters.

  • Search - On the top command menu there is also a Search option, allowing to search any text data on the view -
    1. Click the requested text you wish to search and click the search icon - Only the rows containing the searched text would appear. The searched text would be painted on those rows.
    2. In order to clear the search, click on the X icon near the Search box. 

  • Permanent Filter -
    • Views might be filtered already on openning. This is a case when a permanent filter has been performed on the view.
    • The data that have removed due to permanent filter cannot be displayed on view by user's commands.
    • Changing the permanent filter is available by Admin commands.
