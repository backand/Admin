Durados views allow editing some of the information on the grid itself, without openning the edit dialog.

 

  • Edit On-Grid - Allowing to editing on the grid -  
    1. Operate Edit Mode - make sure that the Edit mode check box (at the right end of the menu row) is checked.
    2. Perform Edit on-grid - the cells that can be edited are those that with borders around them.
    3. Click on the cell - the cell open for editing.
    4. Edit the cell. Each cell can be edited according to its type - free text, choose from dropdown list, date...
    5. When clicking elsewhere outside of the cell, it saves the change.

  • Copy & Paste   - Allowing to perform Copy & Paste actions on-grid.
    This is another option for edit cells on-grid - 
    1. Copy & Paste option is vailable for on-grid editing cells.
    2. Note that cells can be copied only from one cell to another on the same column. It is not possible to paste from one column to the other.
    3. There are several option to perform Copy & Paste:
 

  • copy single cell -
    1. Place the marker on a cell (do not click on it).
    2. Right-click and choose Copy - the cell receives lineal border.
    3. Mark another cell on that column and click Paste.
    4. approve the confirmation message and the value has been copied to that cell.

  • copy several cells   - There are 2 options to mark several cells -
    1. mark close cells - click on the first cell, hold Ctrl and click on the last cell. All the edit on-grid cells between them will be marked.
    2. mark cells that are not side by side, just on the same row - click on 1 cell, hold Ctrl and mark other cells on the line.
    3. All cells that have been selected are marked with border.
    4. Right-click and select Copy - all cells receive a lineal border.
    5. On the row to paste cells, mark the most left cell and click Paste. 
    6. Approve the confirmation message - all copied values would be paste to that row.

  • Copy section -
    1. Mark the most upper & left cell, click Ctrl and mark the most lower and right cell - all the edit on-grid cells betwenn them are marked.
    2. Right-click and select Copy - all cells receive a lineal border.
    3. On the row to paste cells, mark the most left cell and click Paste. 
    4. Approve the confirmation message - all copied values would be paste to that row.

  • Paste the same copied cells to several rows   -
    1. On any of the options above it is possible to paste the copied cells to several rows.
    2. Perofrm the required copy action, as explained above.
    3. When Pasting, mark the left cell on several rows.
    4. Marking several rows is an action similar to marking several cells on row.
    5. Right-click and select Paste.
    6. The confirmation message indicates that the cells are about to be copied to several rows - approve it.
    7. All copied values would be paste to the selected rows.

    1. Besides Durados copy&Paste options, it is possible to mark only the content of the cell and copy it to another cell.
      Text and date cells can be copied, but of course when dealing with drop-downs or multi-select cells, they relay on their own data and copy to them is impossible.
