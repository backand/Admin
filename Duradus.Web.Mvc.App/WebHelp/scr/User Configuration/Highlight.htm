Durados allow marking a specific row, in order to highlight it.

  • Highlight row - Painting a specific row, with a chosen color -
    1. Near the Highlight option (on the upper commands menu) there is a colored box. This box represnt the color chosen to be marked.
    2. Click on that box and choose the color you requestfrom the list.
    3. Mark the selected row.
    4. Click on Highlight - the row would be colored as requested.
