Add, Edit & Delete

Durados system display the following options on all views.

 

There are the most common and allow to perform almost any change on system display:

  • New - Allowing to add new row -
    1. clicking on New  - Add dialog screen opens without any data written on it
    2. User should fill at least the required data.
    3. Insert data to the required fields + data to all the other fields you desire.
    4. Click Save and the new row would be added to the view.

  • Edit - Allowing to edit an existing row -
    1. Select a row and click edit.
    2. The dialog screen display all the existing data of the row.
    3. Insert data to the fields you desire.
    4. Click Save and the row would appear with the new data

  • Multi-Edit - It is possible to edit several rows (or all) on the same action:
    1. Mark a check box of a row, hold Ctrl key down and mark other rows.
    2. After all the rows you want are marked, click on Edit - Edit dialog opens.
    3. The difference between that dialog screen to the Edit (1 row) dialog screen, is the + mark next to some of the fields.
    4. Clicking on the + turn it to - and allow the user to change the value on the field.
    5. Leave the mark as - and save the changes - they will be affective to all selected rows.

  • Duplicate - Another option to add row, but saves the need to insert data to all fields.
    1. Check a row and click duplicate.
    2. The add dialog would open and the fields will display the data from the duplicated row.
    3. Change the requested data and save - the new row would appear.

  • Delete - Allowing to delete an existing row.
    1. check a row to mark it.
    2. Click on Delete - a confiramtion message appear.
    3. Approve the confirmation message - the row removed from the view.
    4. It is also possible to check mark and delete several rows or all - mark them with Ctrl key, as explained on Multi-Edit section.

  • View - Allowing to view rows without changing them.
    1. In case that a view or row are on read-only status (Permissions can allow users of certain roles only to view data), without the options to edit, add ort delete row, the option View will appear instead of Edit at the left of the row.
    2. New, Edit, Duplicate & Delete options will not appear on that view.
