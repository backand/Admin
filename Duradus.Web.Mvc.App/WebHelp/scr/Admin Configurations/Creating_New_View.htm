When adding new table to Durados and to the DB, it should be added to Views:

  • On Admin –Views, click on New

  • New View required the following parameters:

    • Name (actual name of the view – should be identical to name that appears on DB).
    • Display Name (could be any name – it will be the name that will be displayed).

  • In case that there is a dependency between the created table and any existing table, the editable table name should be entered under 'Advanced-Editable Table Name'

  • All the other parameters are not mandatory and can be used for the user's needs.
