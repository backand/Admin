Create New Application

Follow these steps to create new application in Durados

1. Access http://www.durados.net and select New User Registration

2.In the Registration form complete the information by providing your name, email and password and click Submit .

3. Check your email from support@Durados.net and click on the Email Verification in order to complete the registration. Check the SPAM folder in case you dont recive the email.

4. After login Select New Application

5. To connect to an existing database do the follow:

a. Provide App Name which will be come your DNS site name (e.g. Mikecrm --> mikecrm.durados.net)

b. Select App Source as "Existing Database"

c. Provide App Connection string: Server IP or DNS, Catalog name (Database name), Username and Password. You can leave the Authentication Connection and Management Connection empty and the system will create it for you automatically.

d. Use Icon upload button for your own Logo

d. Click on "Save and Close"

6. To Access to the new Application just click on the hyperlink. In the future you can access the application directly by using this URL.
