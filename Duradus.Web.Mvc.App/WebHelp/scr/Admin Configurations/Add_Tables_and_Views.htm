Add Tables and Views

Follow these steps to display the tables and views of the existing database:

1. Access the Application and select from the menu Admin --> Design --> Views

 

2. Select "Add Views" to get all the existing tables and views in the database that were not added yet to the system

3. Select the tables one by one or by Select All when done click on Add. In case of Views you must provide Editable Table which is the table that the view is use to update or add new records

4. In order to see the new added tables and views click on Refresh by selecting Ctrl+F5. All the new tables will be added under New Views menu.
