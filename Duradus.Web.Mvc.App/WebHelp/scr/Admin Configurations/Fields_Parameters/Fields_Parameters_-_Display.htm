The following parameters determine if the field would appear on different status of the view:

  • Hide In View: Checking that parameter will hide it on the main grid, so that field will not appear on the view's grid.
    It can appear on the edit and add dialogs.

  • Hide In Edit: Checking that parameter will hide it on Edit Dialog.
    It can appear on the add dialog and the main grid.

  • Hide In Create: Checking that parameter will hide it on Add Dialog.
    It can appear on the edit dialog and the main grid.

The following parameters determine if the field would allow changes or appear as disabled:

  • Disable In Edit: Checking that parameter will disable it to changes on edit mode. The field would appear on edit dialog but without option to change it.

  • Disable In Create: Checking that parameter will disable it to changes on add mode. The field would appear on add dialog but without option to change it.

  • Disable In Duplicate: Checking that parameter will disable it to changes on duplicate mode. The field would appear on duplicate dialog but without option to change it.


The following parameters determine some aspects of how to display the field:

  • Hide In Filter: Checking that parameter would hide the filter box of the field and will disable the abilitiy to filter by that field.

  • Order: The value on order detemine the order of that field related to other fields on view.
    The fields would appear from left (lowest order values) to right (highest order values).

  • Sortable: Checking that parameter would allow to sort by that field.
