The General parameters are the ones that appear above all the tabs.

Those parameters would be always display when the Edit \ Add dialog is open, no matter which tab have been selected:

  • Name (Required Field) - A ctual name of the field. Should be identical to name that appears on DB. Changing it would affect the connection to the DB.
    Usually shouldn't be changed at all, once it was created.

  • Display Name(Required field) - C ould be any name. It will be the name that will be displayed. This name can change with no harm to the connection to DB.

  • Descrtiption / Tooltip – Free text box which explains the field content. The text would be displayed when standing on the view.
