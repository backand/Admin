Parameters - Description

Description parameters define the tooltip that would be displayed on each of the appearing options, when standing on it:

  • Enter data into one of the text boxes - for example, write 'Add new row' in the New Button Description.

  • Save &Commit, in order to save the canges.

  • When standing (without clicking) on the new option, it will display the description.

  • The following would aply for all the views.

Description is avasilable for the follwoing options:

  • Promote Button Description: Define the description when standing on Promote button.

  • New Button Description: Define the description when standing on New button.

  • Edit Button Description: Define the description when standing on edit button.

  • Duplicate Button Description: Define the description when standing on duplicate button.
